//	ЧТО ЭТО?
// ==================================================
Gulp - это таск-менеджер: приложение, которое используется для автоматизации разработки проекта.



//	УСТАНОВКА:
// ==================================================
1. Установить node.js http://nodejs.org/

2. В коммандной строке выполнить:
npm install gulp -g

2. Из папки builder запустить install-gulp.cmd или в ком. строке выполнить:
npm install gulp gulp-stylus gulp-jade gulp-autoprefixer gulp-connect gulp-plumber gulp-watch

3. Из папки builder запустить run-gulp.cmd 
или в ком. строке выполнить: gulp

4. Верстка будет доступна по адресу http://localhost:8080
В проекте реализованно автообновление сервера при изменении файлов с версткой (Stylus, Jade, JS), подробнее в gulpfile.js



//	ОПИСАНИЕ ФАЙЛОВ И ПАПОК:
// ==================================================
builder ............... Сборщик
	node_modules ...... Cистемная папка с модулями node.js
	gulpfile.js ....... Настройки Gulp
	gulp-install.cmd .. Установка gulp в текущую папку
	gulp-run.cmd ...... Запуск Gulp
site .................. Скомпилированная верстка
source ................ Исходники
	jade .............. jade шаблоны (препроцессор HTML)
	styl .............. stylus шаблоны (препроцессор CSS)



//	ВОЗМОЖНЫЕ ОШИБКИ:
// ==================================================
- При открытии install-gulp.cmd или run-gulp.cmd консоль открывается и закрывается.
	Решение: запустить консоль вручную, ввести команду и прочитать ошибку

- Могут быть проблемы при использовании "[]" в названии папок в пути к проекту.
